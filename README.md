# Spots

An image-sharing webpage that showcases different locations through photo cards.  
Users can view profile information and browse a responsive gallery of images that adapts to different screen sizes.

## Description

Spots is a static web project built to practice clean HTML structure, BEM methodology, and modern CSS layout techniques.  
The page includes a user profile section and an image gallery made up of cards. Each card displays an image and a title, and the layout adjusts smoothly between desktop and mobile screen sizes.

## Tech Stack

- HTML
- CSS
- Responsive Design
- CSS Grid
- Flexbox
- BEM Methodology
